# pathgazeobo

这个仓库现在包含你的 Gazebo 世界文件（`goaero_mission3_v1.sdf`），并补充了一份从 **ArduPilot SITL + Gazebo Sim + ROS 2** 生成 **OctoMap** 的实战指南。

## 快速入口

- 世界文件：`goaero_mission3_v1.sdf`
- OctoMap 管线指南：`docs/octomap_pipeline_zh.md`
- 一键启动示例脚本：`scripts/run_octomap_pipeline.sh`

> 说明：脚本主要用于演示“最小可运行流程”，你需要按自己机器上的实际话题名调整参数。
